# tdd-katas-rust

A collection of code katas in Rust

## Getting Setup

- [Install Rust](https://www.rust-lang.org/learn/get-started)

- [Install VS Code Extensions](https://code.visualstudio.com/docs/languages/rust)

## Check Setup

cd hello-word
cargo run
```

```
